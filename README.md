# Execução do projeto

Na pasta raíz do projeto:

```bash
docker compose up -d
```

O comando acima executa o banco de dados Postgres. 

Para parar a execução

```bash
docker compose down
```



# Deploy

Não há deploy configurado para este projeto, pois este foi feito para execução local pela linha de comando.
